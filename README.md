# Integration-Project-Java / Projeto Integrador – Sistema de Locação
Projeto de gerenciamento para frota de veículos em Java.


## Descrição
Este projeto foi desenvolvido como parte do *Projeto Integrador* do curso de *Tecnologia em Análise e Desenvolvimento de Sistemas*.  
O objetivo é modelar e implementar um *sistema de locação*, simulando um cenário real de negócio, desde o cadastro de clientes e itens até o controle das locações.

O projeto busca integrar conceitos aprendidos ao longo do período, como lógica de programação, organização de dados, regras de negócio e documentação de sistemas.

---

## Objetivo do Sistema
- Automatizar o processo de locação
- Reduzir erros manuais
- Facilitar o controle de clientes, itens e contratos
- Simular um sistema próximo à realidade do mercado

---

## Funcionalidades
- Cadastro de clientes
- Cadastro de itens disponíveis para locação
- Registro de locações
- Controle de datas de retirada e devolução
- Validação de dados
- Consulta de informações
